# Allarme-Arduino
Un semplice sistema di allarme.

Realizzazione di un sistema di allarme con una scheda Arduino UNO R3.

Componenti necessari:
  - UNO R3 Controller Board
  - LCD1602 Module (with pin header)
  - Ultrasonic sensor
  - Passive Buzzer
  - Button
  - 6 resistor 220 Ohm
  - Breadboard
  - Breadboard jumpers
