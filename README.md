# Allarme-Arduino

## Italiano
Un semplice sistema di allarme.

Realizzazione di un sistema di allarme con una scheda Arduino UNO R3.
Premere il bottone per attivare l'allarme. Se il sensore ultrasuoni rileva un movimento e l'allarme è attivo, quest'ultimo scatterà e produrrà un suono.
Per disattivare o spegnere l'allarme utilizzare sempre il bottone.

### Componenti necessari
- UNO R3 Controller Board
- LCD1602 Module (with pin header)
- Sensore ultrasuoni
- Passive Buzzer
- RGB Led
- Bottone
- 6 resistori 220 Ohm
- Breadboard
- Breadboard jumpers

## English
A simple alarm system.

Implementation of an alarm system using an Arduino UNO R3 board.
Press the button to activate the alarm. If the ultrasonic sensor detects movement and the alarm is active, it will trigger and produce a sound.
To deactivate or turn off the alarm, always use the button.

### Required components
- UNO R3 Controller Board
- LCD1602 Module (with pin header)
- Ultrasonic sensor
- Passive Buzzer
- RGB Led
- Button
- 6 resistors 220 Ohm
- Breadboard
- Breadboard jumpers
