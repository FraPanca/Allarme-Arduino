# Allarme-Arduino
Un semplice sistema di allarme.

Realizzazione di un sistema di allarme con una scheda Arduino UNO R3.
Premere il bottone per attivare l'allarme. Se il sensore ultrasuoni rileva un movimento e l'allarme è attivo, quest'ultimo scatterà e produrrà un suono.
Per disattivare o spegnere l'allarme utilizzare sempre il bottone.

Componenti necessari:
  - UNO R3 Controller Board
  - LCD1602 Module (with pin header)
  - Ultrasonic sensor
  - Passive Buzzer
  - RGB Led
  - Button
  - 6 resistor 220 Ohm
  - Breadboard
  - Breadboard jumpers
